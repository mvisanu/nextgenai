# NCR-2024-0412: Landing Gear Retraction Actuator Seal Extrusion

**Report ID:** NCR-2024-0412  
**Date:** 2024-08-02  
**System:** Landing Gear  
**Severity:** Critical  
**Supplier:** PrecisionActuators Inc  
**Part Number:** P/N LGA-320-7710  
**Aircraft:** Airbus A320-271N (NEO), Tail D-AINE  
**Engineer:** Klaus Fischer  

## Defect Description
During post-landing gear retraction test following scheduled maintenance, 
nose landing gear retraction actuator LGA-320-7710 exhibited hydraulic fluid 
bypass past the primary piston seal. Actuator extension force measured at 
42 kN versus required 58 kN minimum — a 28% reduction. Fluid bypass resulted 
in gear-up landing gear doors failing to fully close (3mm gap at door trailing edge). 
No in-service event; fault discovered during ground maintenance functional check.

## Root Cause
Disassembly of actuator (SN PA-20240318) revealed primary piston seal 
(Viton quad-ring P/N VQR-7710) had extruded into the backup ring groove. 
PrecisionActuators Inc production records indicate the wrong backup ring 
clearance was machined on this production lot (LGA-320-7710-LOT-004): 
diametral clearance was 0.31mm against drawing tolerance of 0.08–0.15mm. 
The excessive clearance allowed the seal lip to cold-flow under cyclic hydraulic 
pressure (3,000 psi operating pressure) during 23 cycles of acceptance testing.

## Corrective Action
1. Remove and replace actuator LGA-320-7710 (SN PA-20240318)
2. Quarantine all actuators from LOT-004 — 100% hydraulic proof test before installation
3. PrecisionActuators Inc to perform CNC machine calibration audit on actuator body line
4. Issue SCAR-2024-0412 — require first-article dimensional inspection for all future lots
5. Update QPL entry: add backup ring diametral clearance to incoming dimensional check
