# NCR-2024-0341: Engine Bleed Air Duct Cracking — APU Section

**Report ID:** NCR-2024-0341  
**Date:** 2024-06-07  
**System:** Pneumatic / Bleed Air  
**Severity:** Major  
**Supplier:** ThermoFab Aerospace  
**Part Number:** P/N A380-BAD-2210  
**Aircraft:** Airbus A380-841, Tail F-HPJB  
**Engineer:** Sophie Lefebvre  

## Defect Description
Hairline crack discovered in APU bleed air duct section (station 18, left side) during 
borescope inspection at 12,000-cycle check. Crack length measured at 47mm running 
circumferentially at the flange weld joint. Bleed air temperature at this station 
reaches 380°C during APU ground start operations. No in-flight event reported, 
but crack propagation risk assessed as critical if left unaddressed.

## Root Cause
Metallurgical analysis of failed weld joint (batch WJ-A380-2024-003, ThermoFab 
Aerospace) identified incomplete fusion at root pass. The weld was performed using 
incorrect filler material (AWS ER347 instead of specified AWS ER321 for high-temperature 
cycling applications). The mismatch in thermal expansion coefficients between base 
metal (Inconel 625) and filler caused fatigue crack initiation at the fusion line 
under thermal cycling loads.

## Corrective Action
1. Replace cracked duct section A380-BAD-2210 (SN TF-20240603)
2. NDT inspection (dye penetrant) of all APU bleed duct weld joints on 6 A380 aircraft 
   in fleet with ThermoFab components from batch WJ-A380-2024-003
3. Issue SCAR-2024-0341 to ThermoFab Aerospace — mandatory weld procedure qualification re-audit
4. Revise AMP task card 78-21-12 to add 6,000-cycle borescope check at APU duct flanges
5. Update engineering drawing to specify filler material AWS ER321 explicitly
