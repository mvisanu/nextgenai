# NCR-2024-0519: Flight Control Computer Software Anomaly — Autopilot Disconnect

**Report ID:** NCR-2024-0519  
**Date:** 2024-10-29  
**System:** Flight Controls / Autopilot  
**Severity:** Major  
**Supplier:** FlightSys Technologies  
**Part Number:** P/N FCC-787-SW-v4.1.2  
**Aircraft:** Boeing 787-9, Tail ZK-NZE  
**Engineer:** James Tui  

## Defect Description
In-flight autopilot disconnect event occurred at FL350 during cruise phase. 
Flight crew reported master warning activation and automatic AP disconnect with 
FCC FAULT message displayed. Aircraft deviated 180ft below assigned altitude 
before crew resumed manual control and re-engaged autopilot. Total duration of 
uncommanded deviation: 34 seconds. ACARS maintenance message transmitted: 
FCC1 CH-B MONITOR FAULT. Event replicated twice on subsequent flights before 
software update grounded aircraft.

## Root Cause
FlightSys Technologies engineering review of FCC software v4.1.2 identified 
a race condition in the dual-channel monitor comparison routine. When IRS1 
and IRS2 data update rates diverged by more than 2ms (possible during IRS 
warm-up phase at high altitude), the monitor logic incorrectly flagged a 
cross-channel disagreement and issued an automatic disconnect command. 
The race condition was introduced in v4.1.2 during an unrelated latency 
optimization patch and was not captured by existing hardware-in-the-loop 
test suite, which used synchronized IRS inputs only.

## Corrective Action
1. Update FCC software from v4.1.2 to v4.1.3 (corrected monitor timing logic)
2. Ground 3 aircraft with FCC v4.1.2 installed pending software update
3. FlightSys Technologies to expand HIL test suite — include asynchronous IRS 
   input scenarios with ±5ms delta
4. Issue safety alert to Boeing and all 787 operators — FCC v4.1.2 operational restriction
5. FAA Special Airworthiness Information Bulletin issued; EASA Safety Information Bulletin in progress

## Related Components
- Inertial Reference System IRS-787-1 and IRS-787-2
- Common Data Network CDN-787
- Autopilot Flight Director System AFDS-787
